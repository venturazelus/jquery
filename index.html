Dokumentasi Jquery Plugin Fotorama

ditambahkan code:


dibagian class="fotorama"
data-nav="thumbs"

yang menambahkan gambar thumbnail.

ditambahkan code:

dalam tag class="fotorama"
div class="fotorama" data-transition="crossfade"

data-transition="crossfade" : digunakan untuk animasi crossfade.

ditambahkan code:

dalam tag class="fotorama"
div class="fotorama" data-loop="true"

data-loop="true" : untuk membuat loop.

ditambahkan code:

dalam tag class="fotorama"
div class="fotorama" data-navposition="top"

data-navposition="top" : untuk membuat posisi gambar thumbnail diatas.
